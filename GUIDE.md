# PDFox — User Guide

PDFox is a lightweight PDF viewer with annotation tools built for Windows.

---

## Opening and Closing Files

| Action | How |
|--------|-----|
| Open a PDF | Click **Open** or press `Ctrl+O` |
| Open multiple PDFs | Each opens in a new tab |
| Switch tabs | Click the tab |
| Close the current tab | Click **×** on the tab or press `Ctrl+W` |

---

## Saving

| Action | How |
|--------|-----|
| Save (overwrite original) | `Ctrl+S` — prompts for confirmation |
| Save a copy to a new path | `Ctrl+Shift+S` — updates the current tab to the new path |

An unsaved tab shows **\*** before the filename (e.g. `*document.pdf`).

---

## Navigating

| Action | How |
|--------|-----|
| Scroll | Mouse wheel or scroll bar |
| Previous / next page | `‹` / `›` buttons in the toolbar |
| Jump to a page | Type a page number into the page field and press `Enter` |
| Table of contents | Opens automatically for PDFs that have bookmarks; click any entry to jump to that section; click `▶` to expand children |
| Show / hide contents panel | Click the `☰` button at the top-left of the panel |

---

## Zoom

| Action | How |
|--------|-----|
| Zoom in / out | `Ctrl+scroll` or `Ctrl++` / `Ctrl+-` |
| Fit to page width | `⇔` button or `Ctrl+0` |
| Fit to page height | `↕` button |

---

## Rotation

| Action | How |
|--------|-----|
| Rotate all pages 90° clockwise | Click `↻` |
| Rotate only the currently visible page | `Shift+click ↻` |

---

## Annotation Tools

Select a tool from the toolbar or use the keyboard shortcut.

| Tool | Shortcut | Description |
|------|----------|-------------|
| Select / Pan | `Esc` | Default mode — scroll and read |
| Draw | `D` | Freehand pen stroke |
| Highlight | `H` | Drag over text to highlight it |
| Text box | `T` | Click to place a text label |
| Line | `L` | Draw a straight line |
| Rectangle | `R` | Draw a rectangle |
| Oval | `O` | Draw an oval / circle |
| Arrow | `A` | Draw an arrow |
| Eraser | `E` | Click any annotation to delete it |

**Shift modifier for shapes:** hold `Shift` while dragging to constrain —
rectangles become squares, ovals become circles, lines and arrows snap to 45° increments.

### Text box formatting

Before placing a text box, toggle **B** (bold) and/or **U** (underline) in the toolbar.
The textarea preview reflects these styles so what you see matches the final output.

---

## Colours and Thickness

- Click a colour swatch in the toolbar to change the annotation colour.
- Drag the thickness slider to change stroke width.

---

## Undo / Redo

| Action | How |
|--------|-----|
| Undo | `Ctrl+Z` or `↩` button |
| Redo | `Ctrl+Y` or `↪` button |

Undo/redo history is per-tab and survives tool switches.

---

## Printing

Press `Ctrl+P` or click the `⎙` button.
Annotations drawn on the canvas are included in the printed output.
